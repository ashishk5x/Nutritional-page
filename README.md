# 🥗 Nutrition Label Page

A simple **Nutrition Facts Label** built using **HTML5** and **CSS3**, with a strong focus on **CSS Typography**, spacing, alignment, and layout techniques.

This project recreates the appearance of a real nutrition facts label while practicing fundamental CSS concepts such as typography, flexbox, margins, borders, and responsive spacing.

## 🎯 Learning Objective

The goal of this project was to improve my understanding of **CSS Typography** and recreate a real-world Nutrition Facts label using only HTML and CSS.

During this project, I learned how to:

* Organize content using semantic HTML
* Style text with different font weights and sizes
* Align elements using Flexbox
* Create visual hierarchy with spacing and dividers
* Build clean, readable layouts
